# conte
# コンテ帳 🎬

映像制作の現場に特化した、オフライン対応の字コンテ・絵コンテ作成Webアプリです。
ブラウザ上でサクサク動き、入力した原稿から目安尺を自動計算。さらにVコン（ビデオコンテ）の再生と動画書き出しまでこれ一つで完結します。

## ✨ 主な特徴
* **📝 原稿のタスク消化UI**：ナレーションやセリフをベタ貼りし、カットに割り当てるとストックから消えていく安心設計。
* **⏱ 実務レベルの尺計算**：漢字を2文字分（2モーラ）として重み付け計算し、精度の高い「目安尺」を自動算出します。
* **▶️ Vコン自動生成＆書き出し**：設定したカット尺に合わせて自動再生。さらに `WebM` 形式で録画・ダウンロードでき、Premiere ProやDaVinci Resolveでの編集ガイドとして即利用可能です。
* **📱 オフライン動作（PWA対応）**：一度スマホのホーム画面に追加すれば、飛行機の中や電波のないロケ先でも完全オフラインでサクサク動作します。（データは端末内に自動保存）
* **📄 PDF出力対応**：iPhone標準のプリント機能と連携し、綺麗なフォーマットでコンテをPDF化できます。

## 📱 スマホ（iPhone）へのインストール方法
このアプリは「PWA」として動作します。App Storeからではなく、以下の手順でインストールしてください。

1. iPhoneの **Safari** でこのURLを開く（ChromeやLINE内ブラウザはNG）
2. 画面下の「共有ボタン（四角から上矢印のマーク）」をタップ
3. メニューから **「ホーム画面に追加」** をタップ
4. ホーム画面に追加されたアイコンから起動！
※アップデートがあった際は、アプリを完全に閉じてから開き直すと最新版が適用されます。

## 💡 基本的な使い方
1. **企画情報の入力**：「＋ 新規企画」からプロジェクトを作成。fpsやプロジェクト総尺を設定します。
2. **原稿の流し込み**：「原稿・尺確認」タブのストックエリアに、構成案やセリフをベタ貼りします。
3. **カット割り**：「カードからカットを追加」を押し、カメラワークやコンテ絵（直接お絵描きも可能）を設定します。
4. **Vコン確認**：「Vコン」タブで再生してテンポを確認し、必要に応じて動画として書き出します。
5. **エクスポート**：右上の「EXPORT」ボタンからPDFを出力して共有！

## ⚙️ データ管理について
端末の空き容量不足などでブラウザの保存データが消去されるリスクに備え、トップ画面左上の「データ管理」から、定期的に **全データを保存（JSON）** してバックアップを取ることを推奨します。
