# コンテ帳 🎬

映像制作に特化した、オフライン対応の字コンテ・絵コンテ作成Webアプリです。
ブラウザ上で動き、入力した原稿から目安尺を自動計算。ざっくりとしたVコン（ビデオコンテ）の再生と動画書き出しまで完結します。

## ✨ 主な特徴
* **📝 原稿のタスク消化UI**：ナレーションやセリフをベタ貼りし、カットに割り当てるとストックから消えていく設計。
* **⏱ 原稿から尺計算可能**：漢字を2文字分（2モーラ）として重み付け計算し、「目安尺」を自動算出します。
* **▶️ Vコン自動生成＆書き出し**：設定したカット尺に合わせて自動再生。さらに `WebM` 形式で録画・ダウンロードでき、Premiere ProやDaVinci Resolveでの編集ガイドとして利用可能です。
* **📱 オフライン動作（PWA対応）**：一度スマホのホーム画面に追加すれば、飛行機の中など電波のない場所でも完全オフラインで動作します。（データは端末内に自動保存）
* **📄 PDF出力対応**：iPhone標準のプリント機能と連携しコンテをPDF化できます。

## 📱 スマホ（iPhone）へのインストール方法
このアプリは「PWA」として動作します。App Storeからではなく、以下の手順でインストールしてください。

1. iPhoneの **Safari** でこのURLを開く（ChromeやLINE内ブラウザはNG）
2. 画面下の「共有ボタン（四角から上矢印のマーク）」をタップ
3. メニューから **「ホーム画面に追加」** をタップ
4. ホーム画面に追加されたアイコンから起動！
※アップデートがあった際は、アプリを完全に閉じてから開き直すと最新版が適用されます。

## 💡 基本的な使い方
用途に合わせて2つの作り方ができます。まずは「＋ 新規企画」からプロジェクトを作成してください。

### パターンA：原稿・歌詞・セリフが先にある場合（MV、CMなど）
1. **原稿の流し込み**：「原稿・尺確認」タブのストックエリアに、テキストをベタ貼りします。改行ごとに自動でカードに分割されます。
2. **カット割り**：カードの「＋ この文で新規カット」を押すと、そのテキストが入った状態でカット編集画面が開きます。
3. **詳細の入力**：カメラワークやト書きを設定し、コンテ絵（その場で描画も可能）を追加して保存します。割り当てが終わったテキストはストックから消えます。

### パターンB：映像先行でゼロからコンテを切る場合（PV、アイディア出しなど）
1. **直接カットを作成**：「カット」タブを開き、「＋ カットを追加」を押します。（「📁 シーケンス追加」でAメロ、サビなどのまとまりを作ることも可能です）
2. **詳細の入力**：カットの尺、ト書き、カメラワークを入力し、コンテ絵を描いて保存します。
3. **後からセリフを追加**：必要に応じて、カット編集画面で後からセリフやSEなどを直接書き足すこともできます。

### 共通の仕上げ作業
* **Vコン確認**：「Vコン」タブで再生してテンポや尺感を確認します。「⏺ 録画して書き出す」でWebMデータとして保存できます。
* **エクスポート**：右上の「EXPORT」ボタンからプレビューを確認し、「📄 PDFを作成」で出力・共有します。

## ⚙️ データ管理について
端末の空き容量不足などでブラウザの保存データが消去されるリスクに備え、トップ画面左上の「データ管理」から、定期的に **全データを保存（JSON）** してバックアップを取ることを推奨します。
